Installation Instructions

  1. Install Python 3.13.3 (the version this was created on).
    • If you do not already have Python installed, you will need to restart your computer afterwards.
  2. Download source code from GitHub. (Skip if already downloaded from the website.)
    • On the top right, click the green Code button.
    • Click "Download ZIP".
    • Extract the downloaded ZIP file to a folder where you will use it.
  3. Open your new folder.
  4. Run setupRequirements.bat.
    • Windows may mark this file as unsafe since it is a .bat file. If this happens follow the steps below:
    • Click "More info".
    • Click "Run Anyway".
    • This file simply runs a pip command to download the required Python modules (from requirements.txt) and creates a file called keys.env.
  5. Wait until the command window disappears â€” this means setup is complete.
  6. Open keys.env in Notepad.
  7. Get your API keys. You will need four:
    • Google Maps
    • US Census
    • Gemini
    • FBI key is a public key and is already included
  8. Paste the following into your keys.env file (replace KEY with your actual keys): 
    GoogleMapsKey=KEY
USCensus=KEY
FBI=iiHnOKfno2Mgkt5AynpvPpUQTEyxE77jo1RU8PIv
GEMINI=KEY
  9. Start the program:
    • Double-click run.bat, or
    • From the command line, run: python -u main.py
  10. (Optional) Create a desktop shortcut:
    • Right-click where you want the shortcut (e.g., your desktop).
    • Hover over New > click Shortcut.
    • Navigate to the folder you downloaded and select run.bat.
    • Name the shortcut whatever you'd like.
    • You can now double-click the shortcut to launch the app.
